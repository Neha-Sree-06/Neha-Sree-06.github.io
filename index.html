Smart Walking Stick for Visually Impaired People MAR 2026

Built using Arduino Uno and an ultrasonic sensor to detect obstacles and provide real-time alerts through a buzzer and vibration motor — applying embedded systems, sensor interfacing, and microcontroller programming to improve user safety.

Four-Way Traffic Light Controller OCT 2025

Built a sequential LED switching circuit using a 555 timer astable clock driving a 4017 decade counter, with each counter output stage mapped to a traffic light state. Timing adjusted via potentiometer. Demonstrates discrete IC-based sequential logic design without a microcontroller.

Automatic Battery Charger OCT 2025

Designed an LM317-based adjustable charging circuit with a BC547/BC557 transistor pair for auto-cutoff sensing once the battery reaches full charge. Current limiting set via power resistors (1Ω/2.2Ω/4.7Ω at 2W) for selectable charge rates, with 1N4007 diodes for reverse-polarity protection and an LED charge indicator.
